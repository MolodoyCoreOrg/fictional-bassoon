## 📝 Пошаговая инструкция запуска

### Шаг 1: Подготовка
```bash
cd /workspace/music_bot
```

### Шаг 2: Настройка токенов
Файл `.env` уже создан с вашим токеном. При необходимости отредактируйте:
```bash
nano .env
```

### Шаг 3: Установка зависимостей
```bash
pip install -r requirements.txt
```

### Шаг 4: Запуск бота

**Вариант A - Прямой запуск:**
```bash
python bot.py
```

**Вариант B - Через скрипт с автоперезапуском:**
```bash
./run.sh
```

**Вариант C - Docker:**
```bash
docker build -t gg_loader_bot .
docker run -d --name gg_loader -e BOT_TOKEN="ваш_токен" gg_loader_bot
```

**Вариант D - systemd сервис (для продакшена):**
```bash
sudo cp gg_loader.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable gg_loader
sudo systemctl start gg_loader
```

## 🐛 Решение проблем

### Бот не запускается
1. Проверьте токен в `.env`
2. Убедитесь, что установлены все зависимости
3. Проверьте логи ошибок

### Ошибки при обработке аудио
1. Проверьте установку FFmpeg: `ffmpeg -version`
2. Проверьте права доступа к папке `temp/`

### Inline-режим не работает
1. Включите inline-режим в @BotFather
2. Установите username бота как default helper
